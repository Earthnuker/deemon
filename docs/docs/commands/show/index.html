Show

Table of contents

  1. Monitored Artists
  2. New Releases

Using the show command, you can currently view information pertaining to artists and new releases.

Monitored Artists

Show all currently monitored artists

$ deemon show --artists

New Releases

Show all new releases in last N days

$ deemon show --new-releases 7
